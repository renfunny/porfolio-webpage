Bootcamp Project

Calculator

In this project, I built a functional calculator from scratch. First, I worked on the HTML by structuring everything accordingly and adding all the buttons in their respective rows. Then CSS was used to style the buttons alongside with Flexbox which helped with the layout of the buttons. Lastly, JavaScript was used to add the logic behind the calculator. The challenging part of this project was fully understanding the logic behind the calculator. This was done by using various functions that would call onto other funtions, by having variables store and keep track of certain vaules, and by interacting with the DOM so each button had its own value.
